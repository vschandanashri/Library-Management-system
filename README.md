# Library Management System

## Overview
This project is a Library Management System developed to manage books, users, issue records, and return operations efficiently.

It simplifies the process of maintaining library records digitally.

---

## Features
- Add and remove books
- Issue and return books
- User management
- Search functionality
- Database integration
- Admin operations

---

## Technologies Used
- Java 
- MongoDB
- HTML, CSS, JavaScript


---

## Future Improvements
- Fine payment integration
- Email notifications
- Barcode scanning
- Online book reservation
